## TalentFlow Frontend
This is the official frontend repository for TalentFlow, a modern recruitment management platform. This project was scaffolded as part of Frontend Task-1 to establish a high-performance, scalable development environment.

### 🛠️ Technical Stack
The project utilizes a "bleeding-edge" stack to ensure speed and developer efficiency:

Framework: React 19.2.4 – Utilizing the latest features for improved rendering and hook stability.

Build Tool: Vite 8.0.2 – Powered by the Rolldown engine (Rust-based) for near-instant hot module replacement (HMR).

Styling: Tailwind CSS v4.2.2 – Using the new CSS-first engine and @tailwindcss/vite plugin for zero-runtime overhead.

Routing: React Router 7.13.2 – Handling complex navigation with a robust, data-aware routing system.

Linting: ESLint 9.39.4 – Configured with Flat Config for consistent code quality across the team.

### 📁 Project Structure
The scaffolding is organized to support modular growth:

src/components/: Reusable UI elements.

src/pages/: Main application views.

src/main.jsx: The application entry point as defined in index.html.

eslint.config.js: Custom linting rules, including a specific regex to ignore unused variables starting with capital letters or underscores.

### 🚀 Getting Started
Install Dependencies:

npm install 

Start Development Server:

npm run dev 

Build for Production:

npm run build
